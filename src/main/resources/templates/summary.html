Gráfico de % Indisponibilidad por Día

Seleccione una fecha y haga clic en un punto para ver el detalle del reporte o sobre una hora para filtrar ±30 min.

Manual de Uso

  • Selector de fecha: Use el calendario para seleccionar un día con reportes disponibles.
  • Gráfico interactivo: Cada punto representa un reporte. Puede hacer clic en el punto para ver el detalle completo del reporte.
  • Filtro por hora: Al hacer clic en la etiqueta del eje X (hora), se filtrará el gráfico mostrando datos dentro de ±30 minutos desde esa hora.
  • Botón "Volver al resumen diario": Aparece tras aplicar un filtro horario y permite regresar al resumen completo del día seleccionado.
  • Interactividad visual: Las etiquetas del eje X cambian el cursor al pasar sobre ellas para indicar que son clickeables.
